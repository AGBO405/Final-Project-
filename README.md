Project Overview

This project explores the question: What is the best neighborhood in Pittsburgh? Using datasets from the Western Pennsylvania Regional Data Center (WPRDC), we created a data-driven method to compare neighborhoods and determine which one ranks the highest based on monthly criminal activity, housing, and finally employment.

Each of us analyzed a different dataset and created their own sub-metric. These sub-metrics were then combined into a single overall score to determine the “best” neighborhood.

Dataset(s) Used

Monthly Criminal Activity Dataset (WPRDC)

Housing 2010 Dataset (WPRDC)

Employment 2010 (WPRDC)


Repository Overview

This repository contains all code, data, and analysis used in our project to determine the best neighborhood in Pittsburgh.
